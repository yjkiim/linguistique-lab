# NLP & Linguistics Lab

This repository contains small computational linguistics projects developed as part of my self-learning in Python and my academic background in Linguistics (Université Paris Cité).

The goal is to explore how linguistic structures can be analyzed using basic computational methods.

## Focus Areas
- Corpus-based linguistics
- Lexical analysis
- Basic natural language processing (NLP)
- Quantitative linguistic methods

## Projects

### 1. Word Frequency Analysis
Basic lexical frequency analysis of a text corpus.

### 2. Sentence Length Analysis
Comparison of sentence structures across different texts.

## Tools
- Python (beginner level)
- Standard libraries only (no external NLP frameworks)

## Academic Context
This work is aligned with my interest in theoretical and computational linguistics, particularly within the LTEI (Linguistique Théorique, Expérimentale et Informatique) program.
