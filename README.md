# Linguistics & NLP Lab

This repository contains computational linguistics projects developed as part of my academic training in **Sciences du langage (Université Paris Cité)** and my personal exploration of **Python-based natural language processing (NLP)**.

The main objective is to investigate how linguistic structures can be analyzed using basic computational and corpus-based methods.

---

## Research Orientation

This work focuses on:

- Corpus-based linguistics  
- Lexical and syntactic analysis  
- Multilingual language comparison  
- Quantitative approaches to linguistic data  
- Introductory natural language processing (NLP)

---

## Research Question

How can computational methods help reveal structural and lexical differences across languages, and what do these differences tell us about linguistic typology?

---

## Projects

### 1. Word Frequency Analysis
Lexical frequency analysis of a text corpus using Python.

### 2. Sentence Length Analysis
Comparison of syntactic complexity across different texts using sentence length as a proxy.

### 3. Multilingual Corpus Comparison
Cross-linguistic analysis of Korean, French, and English corpora to explore lexical distribution and typological differences.

---

## Tools

- Python (beginner level)
- Standard libraries only (re, collections)
- No external NLP frameworks

---

## Academic Context

This project is aligned with my studies in linguistics and my interest in the **LTEI (Linguistique Théorique, Expérimentale et Informatique)** program, which combines theoretical linguistics with computational and experimental approaches.

---

## Goal

To bridge theoretical linguistics with computational methods through small-scale corpus analysis projects.
