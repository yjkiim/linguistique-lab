# Project 1: Word Frequency Analysis

## Objective
To analyze lexical frequency in a text corpus using basic Python tools.

## Methodology
- Text preprocessing (lowercasing, tokenization)
- Word segmentation
- Frequency computation using Python Counter
- Extraction of most frequent lexical items

## Linguistic Perspective
This project explores:
- Lexical distribution in natural language
- Corpus-based linguistic analysis
- Quantitative approach to vocabulary usage

## Tools
- Python (standard library only)

## Output
The program outputs the top 10 most frequent words in the corpus.
