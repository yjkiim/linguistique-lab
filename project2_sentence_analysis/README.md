# Project 2: Sentence Length Analysis

## Objective
To compare sentence structure complexity across different texts using sentence length as a proxy.

## Methodology
- Sentence segmentation using regex
- Word counting per sentence
- Statistical comparison of average sentence length

## Linguistic Perspective
This project explores:
- Syntactic complexity (proxy measurement)
- Quantitative stylistics
- Structural variation in language

## Tools
- Python (standard library)
